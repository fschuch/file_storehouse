What’s New#

The format is based on Keep a Changelog, and this project adheres to Intended Effort Versioning (EffVer for short). Release notes are automatically recovered from GitHub by sphinx-github-changelog.

v0.5.1#

Released on 2025-07-11 - GitHub - PyPI

Other Changes

  • Apply project management changes from Wizard-template v0.3.1 by @fschuch in #25
    • Updated GitHub Actions workflows to pin action versions, adjust permissions, and refine triggers and source paths for improved security and reproducibility.
    • Raised minimum supported Python version to 3.9 and added Python 3.13 support in project configuration.
    • Improved project metadata, dependency management, and source directory references in configuration files.
    • Introduced automated dependency updates and changelog categorization via GitHub Actions and Dependabot.
    • Added new documentation and contribution guides using Jupyter Book, with improved structure and navigation.
    • Added new CI/CD workflows for testing, building, releasing, and documentation deployment.
    • Migrated project configuration from Poetry to Hatch and PEP 621 standards.
    • Removed legacy configuration files, old workflows, and outdated changelogs.
    • Improved README formatting and documentation presentation.
